Project Name: VerseTrack

Overview
VerseTrack is a comprehensive, gamified Bible reading tracker designed to keep users engaged and consistent in their faith journey. It features flexible reading plans, GitHub-style progress heat maps, AI-powered chapter summaries, and communal "Reading Circles" to foster group accountability.

Getting Started

•	Clone: git clone https:[//github.com/YshuaManzano/versetrack](https://github.com/YshuaManzano/BibleTracker).git

•	Install Frontend: cd app/frontend && npm install

•	Install Backend: cd app/backend && pip install -r requirements.txt

•	Environment: Create a .env file based on .env.example in both backend and frontend directories (include your Emergent LLM key).

•	Run Backend: uvicorn server:app --reload

•	Run Frontend: npx expo start
